## Deployment

### Podman Compose

```yaml
services:
  headscale:
    image: ghcr.io/daemonless/headscale:latest
    container_name: headscale
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
    volumes:
      - "/path/to/containers/headscale:/config"
    ports:
      - 8080:8080
    restart: unless-stopped
```
